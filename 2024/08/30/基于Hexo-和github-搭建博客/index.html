基于Hexo 和github 搭建博客

2024/8/30

1.安装git和Node.js

  • 打开Git Bash输入

  • 创建SSH私钥

Host github.com
HostName ssh.github.com # 这是最重要的部分
User git
Port 443
PreferredAuthentications publickey
IdentityFile ~/.ssh/id_rsa

//测试ssh链接
